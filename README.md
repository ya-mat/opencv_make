# opencv_make

This is the shell script to make libraries of OpenCV-4.5.1 using g++.

## How to use

```
git clone https://github.com/ya-mat/lapack_etc_make.git
cd lapack_etc_make
sh make.sh
```

## Dependency

You must have already installed these softwares below.

- libgtk2.0-dev
- pkg-config
- libavcodec-dev
- libavformat-dev
- libswscale-dev
- Eigen
